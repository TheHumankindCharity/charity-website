## 🚀 Getting Started

To run the project locally:

1. **Clone the repository**

    ```bash
    git clone https://github.com/TheHumankindCharity/charity-website.git
    ```

2. **Navigate into the directory**

    ```bash
    cd charity-website
    ```

3. **Install dependencies**

    ```bash
    npm install
    ```

4. **Start the development server**
    ```bash
     npm run dev
    ```
